# NVTube v3.0: Lightning Fast YouTube Search
**Developer:** Volkan Özdemir Yazılım Hizmetleri
**Website:** [volkan-ozdemir.com.tr](https://www.volkan-ozdemir.com.tr)
**Contact:** bilgi@volkan-ozdemir.com.tr

NVTube is a powerful, professional, and lightweight add-on for NVDA that allows you to search for YouTube videos instantly and play them in a streamlined, ad-free environment.

## Key Features
* **Global Accessibility:** Access the search engine from anywhere by pressing `NVDA+Shift+Y`.
* **Professional Speed:** Find and play the most relevant video result in seconds.
* **Ad-Free Playback:** Opens videos in a special web-view to ensure a clean listening and viewing experience.
* **Dual-Language Interface:** The user interface supports both **Turkish** and **English** for global accessibility.
* **Developer Support:** Integrated secure donation link via PayTR to support ongoing development for the visually impaired community.

## How to Use
1. Press `NVDA+Shift+Y`.
2. Enter the video name, song, or artist in the search box.
3. Press **Enter** or click the **"Ara / Search"** button.
4. Your video will automatically start playing in your default browser instantly.
